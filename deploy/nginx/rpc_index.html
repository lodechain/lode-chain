LODE

Public RPC API

The LODE Testnet Public RPC API is active and ready to serve your requests.

curl -X POST https://rpc.lodechain.org/json_rpc \ -d '{"jsonrpc":"2.0","id":"0","method":"get_info"}' \ -H 'Content-Type: application/json'

* Rate limited to 10 requests per second.
* Only safe methods are exposed via --restricted-rpc.
