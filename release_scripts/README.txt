LODE TESTNET - GETTING STARTED

Welcome to the LODE Testnet! 

1. TO RUN THE NODE:
   Double-click "run_node.bat". This will start syncing the blockchain.
   You can mine LODE by typing "start_mining <your_wallet_address> <number_of_threads>" inside the node window.

2. TO CREATE A WALLET:
   Double-click "create_wallet.bat". It will connect to the remote RPC so you don't even need to finish syncing your node to create a wallet.
   * Make sure to save your 25-word seed phrase safely!

3. TO OPEN YOUR WALLET:
   Double-click "open_wallet.bat".

For more information, visit https://lodechain.org
